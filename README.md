# ryusei_cafe

Sistema de um manga café desenvolvido em java para a disciplina de Programação Orientada a Objetos.


## Requisitos para rodar:
* JVM (Java Virtual Machine) instalado na sua máquina.

## Modos de execução:

### Pela GUI (ainda em desenvolvimento):
1) Compilar o código:
    ```bash
    javac RyuseiCafeGUI.java
    ``` 

2) Executar o programa:
    ```bash
    java RyuseiCafeGUI
    ```

### Pela linha de comando(já implementado corretamente):

1) Compilar o código:
    ```bash
    javac ryusei_cafe.java
    ```

2) Executar o código:
    ```bash
    java ryusei_cafe
